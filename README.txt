BIWD2208 – Web Design Class Test 2 Website

Business Name:
Basotho Global Connect

How to open:
1. Open the folder BIWD2208_Website_Test2.
2. Double-click index.html.
3. Test the menu links: Home, Services, Store/Booking, Blog, Essay, Contact.
4. Test Store/Booking by adding products/services to cart and pressing Checkout.

How to host:
Option 1: Netlify
1. Go to https://www.netlify.com/
2. Create an account or sign in.
3. Drag and drop this whole folder into Netlify Deploys.

Option 2: GitHub Pages
1. Create a GitHub repository.
2. Upload all files and folders.
3. Go to Settings > Pages.
4. Select the main branch and save.

Important:
This is a static website version that you can host directly. For WordPress submission, you can copy the same content into WordPress pages and use WooCommerce for the store.
